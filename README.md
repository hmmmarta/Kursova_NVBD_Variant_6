# Kursova_NVBD_Variant_6
Курсова робота з предмету "Надвеликі бази даних". Варіант 6 – Поліклініка

## Тема
Розробка та аналіз надвеликої бази даних для предметної області "Поліклініка"
з використанням Microsoft SQL Server (SSIS, SSAS, SSRS)

## Автор
ПІБ: Грищечко Марта  
Група: ПМ-41
Варіант: 6 – Поліклініка  

## Опис проєкту
У даному проєкті реалізовано повний цикл побудови надвеликої бази даних:
- реляційна БД
- Data Warehouse
- ETL-процеси
- OLAP-куб
- аналітичні звіти

## Структура репозиторію
- Database – SQL-скрипти створення БД
- SSIS_Projects – ETL пакети
- SSAS_Projects – OLAP куб
- SSRS_Projects – аналітичні звіти
- Documentation – пояснювальна записка

## Використані технології
- Microsoft SQL Server
- SQL Server Integration Services (SSIS)
- SQL Server Analysis Services (SSAS)
- SQL Server Reporting Services (SSRS)

## Інструкція запуску
1. Виконати SQL-скрипти з папки Database
2. Запустити ETL пакети SSIS
3. Обробити OLAP-куб
4. Переглянути звіти в SSRS
